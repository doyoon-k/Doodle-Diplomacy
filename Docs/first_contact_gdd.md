## 1. 게임 개요

### 1.1 컨셉

외계인과 언어로 소통할 수 없는 상황에서, **그림으로 의도를 전달해 지구의 운명을 결정짓는 비언어적 협상 게임.**

한 런(run)은 외계인이 지구에 도착해서 떠날 때까지의 짧은 교섭 과정으로 구성된다.

### 1.2 핵심 키워드

- 비언어 소통 / 그림 기반 협상
- 의도 설계와 시각적 구성
- 외계인 성향 추리
- 지구의 운명이 걸린 높은 스테이크

### 1.3 플레이 경험 목표

플레이어는 드로잉 실력이 아닌 **의도 설계와 해석 추리**를 통해 게임을 풀어나간다.
첫 접촉 특유의 신비감과 긴장감을 경험하며, 매 라운드의 선택이 인류의 운명을 좌우한다는 무게를 느낀다.

## 2. 핵심 메카닉

### 2.1 장면 구성

매 라운드마다 외계인이 두 개의 사물 이미지를 제시한다.
플레이어는 이 두 사물을 이용해 **"무엇을 할 수 있는가"** 를 표현하는 하나의 장면을 구성해 제출해야 한다.

> 예쁜 그림을 그리는 것이 아니라, **두 사물 사이의 관계와 의도를 어떻게 보이게 하느냐**가 핵심이다.

### 2.2 입력 방식

| 방식 | 설명 |
|------|------|
| **스티커 배치** | 외계인이 제시한 사물 이미지를 스티커처럼 바로 사용하여 배치 |
| **직접 드로잉** | 플레이어가 원하면 직접 그려서 보충 가능 |

**설계 원칙:**
생성형 파이프라인 없이, 플레이어의 의도와 제출물 사이 거리를 최소화하는 방향으로 설계한다.

> **핵심 원칙:** 플레이어의 선택과 구성 자체가 게임플레이의 중심이어야 한다. 생성기를 돌리는 느낌이 되어서는 안 된다.

## 3. 외계인 성향 체계

### 3.1 2축 대립 구조

외계인의 성향은 **2개의 독립적인 대립축**으로 구성된다.
각 외계인은 이 축들에서 특정 방향을 선호하는 고유 성향을 가진다.

```
축 1: [방향 A] ←————————→ [방향 B]
축 2: [방향 C] ←————————→ [방향 D]
```

### 3.2 해석 로직

```
플레이어 장면 제출
    ↓
VLM — 장면을 텍스트로 해석
    ↓
LLM — 각 성향 축에 대한 만족도를 카테고리로 판정
    ↓
점수 산출 (축당 -2 ~ +2)
```

**점수 범위:** 라운드당 성향 축 2개 각각 판정되므로, 한 라운드의 점수 범위는 **-4 ~ +4**

## 4. 3단계 피드백 레이어

플레이어에게 전달되는 피드백은 세 개의 서로 다른 채널을 통해 **순차적으로** 제공된다.
각 레이어는 서로 다른 정보 품질과 해석 여지를 가진다.

### 4.1 레이어 1 — 부관 VLM 프리뷰

| 항목 | 내용 |
|------|------|
| **타이밍** | 제출 전 |
| **기능** | 플레이어의 장면을 VLM으로 해석해 "아마도 ~~를 그리신 것처럼 보입니다"라는 의견 제공 |
| **역할** | 의도와 실제 시각적 인상의 간극을 사전에 확인 → 억울함 방지 |

> **설계 포인트:** 부관은 "인간 시각"으로 해석한다. 외계인은 인간과 다른 인지 체계로 보므로, 부관의 해석과 외계인의 반응이 다를 수 있다. 이 간극이 외계인의 이질감을 체감하는 지점이다.

### 4.2 레이어 2 — 외계인 제스처 반응

| 항목 | 내용 |
|------|------|
| **타이밍** | 제출 직후 |
| **기능** | 표정, 몸짓 등 비언어적 제스처로 즉각적인 감정 반응 전달 |
| **역할** | 해석의 여지가 있어 추리의 재미 유지 |
| **보조** | 부관이 옆에서 나레이션으로 "~하고 있네요"라고 거들어 반응 해석을 보조 |

**구현:** 3D 애니메이션으로 표현하며, 모델은 **Meshy.ai**로 생성, 모션 캡처는 **FreeMocap(오픈소스)** 으로 검토 예정이다.

### 4.3 레이어 3 — 과학장교 해석기 (텔레파시 조각)

| 항목 | 내용 |
|------|------|
| **타이밍** | 제스처 반응 후 |
| **기능** | 외계인의 텔레파시를 불완전한 해석기로 부분 번역해 터미널에 표시 |
| **역할** | 제스처에서 읽은 추론과 교차 검증 가능 → 성향 추리 지원 |

> **설계 포인트:** 해석기의 출력량은 라운드마다 들쑥날쑥하게 변동한다. 어떤 라운드에서는 노이즈만 나오고, 선명하게 잡힌 라운드의 정보가 더 값지게 느껴지도록 한다.

### 4.4 피드백 레이어 요약

| 레이어 | 타이밍 | 채널 | 해석 여지 |
|--------|--------|------|-----------|
| 1. 부관 VLM 프리뷰 | 제출 전 | 텍스트 의견 | 낮음 (명확) |
| 2. 외계인 제스처 | 제출 직후 | 3D 애니메이션 | 높음 (모호) |
| 3. 해석기 출력 | 제스처 후 | 터미널 텍스트 | 중간 (불완전) |

## 5. 라운드 루프

### 5.1 기본 흐름

```
① 사물 제시
   └─ 외계인이 두 개의 사물 이미지를 제시한다.

② 장면 구성
   └─ 플레이어가 두 사물의 관계와 의도를 설계해 장면을 만든다.

③ 부관 프리뷰
   └─ 부관 VLM이 해석을 제공한다.
      플레이어는 수정할지, 바로 제출할지 선택한다.

④ 제출 & 제스처 반응
   └─ 외계인이 표정과 몸짓으로 반응한다. 부관이 나레이션으로 보조한다.

⑤ 해석기 출력
   └─ 과학장교의 해석기가 텔레파시 조각을 터미널에 표시한다.

⑥ 추리 & 준비
   └─ 플레이어가 반응을 분석하고, 다음 라운드를 준비한다.
```

### 5.2 라운드 내 감정 리듬

각 단계의 감정 톤이 다르므로 반복감이 줄어든다:

| 단계 | 감정 톤 |
|------|---------|
| 구상 단계 | 창의적 긴장감 |
| 부관 프리뷰 | 확인과 조정 |
| 외계인 반응 | 긴장과 해석 |
| 해석기 출력 | 분석과 학습 |

## 6. 런 구조

### 6.1 런 길이

한 런은 **5라운드**로 구성된다.

### 6.2 점수 계산

| 항목 | 값 |
|------|-----|
| 라운드당 점수 범위 | -4 ~ +4 (축 2개 × 카테고리 점수) |
| 마지막 라운드 가중치 | ×1.5 (마지막 인상이 결말을 뒤집을 수 있는 긴장감) |
| 누적 점수 범위 | -22 ~ +22 |

### 6.3 매크로 피드백

외계인의 태도가 런 진행에 따라 단계적으로 변화하며, 이것이 텍스트 없이 현재 상태를 전달하는 매크로 피드백 역할을 한다.

```
긍정적 흐름: 경계 → 호기심 → 개방
부정적 흐름: 경계 → 불안 → 실망 → 적대
```

## 7. 결말 분기

누적 점수에 따라 **4개의 결말**로 분기된다.
스테이크가 높을수록 플레이어의 매 라운드 선택에 무게가 실린다.

| 누적 점수 구간 | 결말 |
|--------------|------|
| 높음 | (우호적 결말) |
| 중상 | (긍정적 결말) |
| 중하 | (부정적 결말) |
| 낮음 | (적대적 결말) |

> *구체적 점수 구간 및 결말 내용은 추후 확정 예정*

## 8. 사물 풀 & 제시 로직

### 8.1 방향 (검토 중)

| 옵션 | 방식 | 설명 |
|------|------|------|
| **옵션 A** | 프리셋 | 재미있는 조합들을 사전에 대량 제작하고 랜덤 추출 |
| **옵션 B** | LLM 생성 | LLM을 통해 매 런마다 어울리는 조합을 동적으로 생성 |

### 8.2 설계 원칙

- 너무 자명한 관계를 가진 조합은 피한다 (퍼즐성 저하)
- 너무 무관한 조합도 피한다 (의미 구성 불가)
- **다양한 관계 해석이 가능한 조합이 이상적**
