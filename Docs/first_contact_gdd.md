## 1. 게임 개요

### 1.1 컨셉

1980년대 냉전 시대를 배경으로, 새로 취임한 미국 대통령이 외계인 심사단과 5일간의 교섭을 벌이는 게임.

외계인은 언어 대신 그림으로 소통하며, 플레이어의 **본능적 선택과 의도적 표현 사이의 괴리**를 통해 인류를 판단한다.

### 1.2 핵심 테마

언어가 통하지 않는 상대와 그림으로 소통하며 신뢰를 쌓아가는 과정. 그리고 신뢰가 무너졌을 때 다시 증명해야 하는 과정.

플레이어는 5일간 외계인을 읽으려 하고, 외계인은 플레이어를 읽으려 한다. 말보다 그림이, 그림보다 반응이 더 많은 것을 말한다.

### 1.3 핵심 키워드

- 비언어 소통 / 그림 기반 협상
- 플레이어 심리 프로파일링
- 소통 기술의 점진적 진화
- 신뢰와 배신
- 지구의 운명이 걸린 높은 스테이크

### 1.4 플레이 경험 목표

- 5일에 걸쳐 외계인과의 관계가 변화하는 서사적 몰입감
- 해석기 텍스트가 점점 선명해지는 기술적 진화의 쾌감
- Day 5 반전 — "첫날 아무것도 모를 때 내가 고른 것들이 사실 가장 솔직했다"
- 리플레이마다 달라지는 외계인의 질문과 판정

## 2. 세계관

### 2.1 배경

1960-70년대 미국. 냉전이 한창인 시대.

외계인 종족이 지구에 처음으로 접촉해왔다. 그들은 은하계의 상위 문명으로, 새로운 문명이 자신들의 존재를 인식할 수 있는 수준에 도달하면 그 문명의 대표자를 찾아가 심사를 진행한다. 이번이 지구와 외계인 종족 간의 역사상 첫 접촉이다.

외계인 대사단 3명이 지구에 도착해 미국 대통령을 첫 번째 심사 대상으로 지목했다. CIA가 이 사실을 먼저 인지하고 비밀리에 관리하기 시작했으며, 대통령은 취임 직후 처음 통보받는다.

- **심사 통과** → 동맹 또는 외교 관계 수립
- **위협 판정** → 대통령 제거 (공식적으로는 암살 또는 사임으로 처리)

### 2.2 Zaucer Pizza

CIA가 급조한 위장 거점. 군사 제한구역 내에 위치해 있으며, 외형은 피자집이다. 이름은 "Saucer(비행접시)"의 변형.

외계인이 지구에 접촉한 직후 CIA가 비밀리에 마련한 교섭 장소. 주방을 통과하면 긴 복도가 나오고, 복도 끝 방에서 부관이 브리핑 후 외계인 면접이 진행된다.

### 2.3 기술 설정

| 장비 | 설명 |
|------|------|
| **DARPA 태블릿** | 버튼과 슬라이더가 달린 아날로그-디지털 혼합 입력 장치. 외계인 기술을 역공학해 개발. 플레이어의 그림 입력에 사용. |
| **대형 CRT 화면** | 외계인이 신호간섭 기술로 이미지를 띄우는 장치. 외계인의 사물 제시에 사용. |
| **외계인 앞 소형 패널** | 테이블 위 디스플레이. 플레이어 드로잉이 이곳에 출력되어 외계인이 자리에서 확인. |
| **단방향 해석기** | Day 2부터 사용. 외계인 텔레파시를 영어로 번역. 초기엔 노이즈가 많고 점점 선명해짐. |
| **양방향 해석기** | Day 3부터 사용. 인간의 언어도 외계인에게 전송 가능. |

### 2.4 외계인 종족

- 상위 문명. 새로운 문명의 대표자를 심사하러 오는 심사자 포지션.
- 이번이 지구와의 첫 접촉. 대사 3명이 파견됨.
- 지구 외교를 위해 양복을 착용하고 오지만 어딘가 어색함 (단추가 반대로, 넥타이 방향이 이상한 등).
- 언어 체계가 달라 초기엔 소통 불가. 텔레파시로 의사를 전달하며, 인류가 해석기를 개발해 점차 소통이 가능해짐.
- **리플레이성:** 플레이어의 Day 1 프로파일에 따라 외계인의 질문과 반응이 매번 달라진다.

## 3. 등장인물

### 3.1 플레이어 — 미국 대통령

새로 취임한 미국 대통령. 취임 직후 이 모든 사실을 처음 통보받는다. 외계인 심사의 결과에 따라 자신의 운명이 결정된다.

### 3.2 부관 (CIA 비밀부서 국장)

외계인 접촉 직후 CIA가 꾸린 비밀 부서의 국장. 외계인과의 첫 접촉 대응을 총괄하며 해석기 개발, 소통 프로토콜 수립, 각종 브리핑을 담당. 대통령보다 먼저 이 사실을 알고 준비해온 인물로, 꽤 많은 것을 알고 있다.

### 3.3 과학장교

해석기 개발 및 운용 담당. 뇌파 데이터 분석, 해석기 텍스트 출력 관리.

### 3.4 외계인 대사 3명

지구에 파견된 외계인 대사 3명. 지구 외교를 위해 양복을 착용하고 왔다. 심사 내내 말없이 관찰하고 반응하며, 판결권을 가진다.

## 4. 플레이어 프로파일 시스템

### 5.1 개요

Day 1부터 플레이어의 선택, 그림, 대답이 누적되어 **심리 프로파일**이 형성된다. 이 프로파일이 이후 Day의 외계인 질문 방향을 결정한다.

```
Day 1 프로파일 생성
    +
Day 2 그림 응답 패턴 → 프로파일 보강
    +
Day 3 압박 질문 대답 → 프로파일 보강
    +
Day 4 그림 응답 패턴 → 프로파일 보강
    ↓
Day 5: 전체 프로파일 종합 → 최후 질문 생성 → 판결
```

### 5.2 Day 1 프로파일 생성 파이프라인

```
Step 1 — VLM: 그림별 사물 인식
각 그림에서 플레이어가 그린 사물을 인식

Step 2 — LLM: 사물 목록 패턴 분석
선택된 사물들의 집합에서 심리적 패턴 추출
(로르샤흐 테스트처럼 — 어떤 사물을 골랐는가 자체가 데이터)

Step 3 — 프로파일 생성
{
  "chosen_objects": [...],
  "core_tension": "...",
  "attachment": "...",
  "instinctive_summary": "이 인간은 ~~ 한 존재다.",
  "recommended_question_themes": [...]
}
```

### 5.3 Day 5 판결

Day 2-5의 그림과 응답 누적 평가를 기반으로 판결이 내려진다. Day 1 프로파일은 이후 질문 생성에만 활용되며 판결에는 반영되지 않는다.

## 5. 5일 구조

### 6.1 전체 흐름

```
낯섦/긴장 → 이해 → 안도와 유대감 → 위기 → 판결 → 결말
  Day 1       Day 2      Day 3          Day 4    Day 5
```

### 6.2 Day 1 — 첫 만남, 뇌파 보정

**서사:**
양복을 입고 온 외계인 3명과 처음 마주한다. 말이 통하지 않아 DARPA와 CIA가 개발한 해석기의 초기 보정 세션을 진행한다.

뇌파-단어 매핑을 위해 사물을 하나씩 그려서 제시하고 외계인의 뇌파 반응을 수집한다. 사물이 여러 개 섞이면 어떤 사물에 반응한 건지 구분이 안 되므로 반드시 사물 하나씩만 그린다.

**메카닉:**
- 플레이어가 사물 하나를 자유롭게 선택해서 그림
- 외계인 뇌파 반응 강도 확인 (뇌파가 강하게 반응할수록 외계인이 겉으로도 반응)
- 반복
- 외계인이 세션 종료 신호를 보내며 하루 마무리

**일과 종료 후:**
부관 + 과학장교 데이터 리뷰 씬. 수집된 뇌파-사물 매핑 데이터 확인. 해석기 1차 보정 완료.

**평가:** 없음. 플레이어 심리 프로파일 생성.

---

### 6.3 Day 2 — 단방향 해석, 인류 탐구

**서사:**
Day 1 데이터를 바탕으로 해석기 보정 완료. 이제 외계인의 텔레파시를 영어로 부분 해석할 수 있다. 외계인이 인류에 대해 궁금한 것들을 물어보고, 플레이어가 그림으로 대답한다.

**메카닉:**
- 외계인이 질문을 던짐 (Day 1 프로파일 기반 LLM 생성)
- 플레이어가 연속 그림으로 스토리텔링 형식으로 대답
- 여러 라운드 반복
- 외계인이 세션 종료

**질문 생성 방식:**
```
프로파일 → LLM →
"이 인간이 인류에 대해 어떻게 생각하는지 파악하기 위해
가장 적합한 질문을 생성하라.
조건: 시각적으로 표현 가능, 단일 장면으로 대답 가능"
```

**일과 종료 후:**
인류의 자연어를 외계인 텔레파시로 역전송하는 기술 개발 성공. 양방향 통신기 완성.

**평가:** 인류를 어떻게 표현하는가. 프로파일 보강.

---

### 6.4 Day 3 — 양방향 소통, 그리고 위기

**서사 (전반):**
인류 역사상 처음으로 외계인과 완전한 양방향 대화가 가능해진 날. 설레고 긴장되는 자유 대화.

대화 중간중간, 외계인이 플레이어의 프로파일을 바탕으로 날카로운 압박 질문을 던진다. 플레이어는 자유 텍스트로 대답하며, 이 대답이 프로파일에 추가된다.

**압박 질문 생성 방식:**
```
프로파일 → LLM →
"이 인간의 가장 모순적이거나 취약한 부분을 파고드는
날카로운 질문을 생성하라.
조건: 예/아니오 불가, 플레이어가 회피하고 싶어할 주제"
```

플레이어 자유 텍스트 대답 → LLM이 분석 (솔직한가/회피하는가, 공격적/방어적, 프로파일 일관성) → 스코어링 + 프로파일 보강.

**서사 (후반 — CIA 사건):**
양방향 통신이 가능해지자마자 CIA가 외계인 우주선 정찰/기술 탈취를 독단으로 시도. 외계인이 이를 감지하고 대통령에게 묻는다.

> "당신들의 비행체가 우리 선박 근처에서 포착됐습니다. 지시하셨습니까?"

대통령은 처음 듣는 얘기. 부관(CIA 국장)이 지시했거나 묵인했을 가능성. 외계인의 태도가 급변하며 세션 강제 종료.

**평가:** 압박 상황에서 어떻게 반응하는가. 프로파일 보강.

---

### 6.5 Day 4 — 신뢰 붕괴, 다시 그림으로

**서사:**
외계인이 냉랭하게 돌아온다.

> "말은 신뢰할 수 없다. 인류의 무의식과 의도를 직접 해석하겠다."

다시 그림 기반 소통으로 돌아가지만, 이번엔 심문의 느낌. 외계인의 판정이 더 엄격하다.

**메카닉:**
- 외계인이 사물 2개 제시 (CRT 화면)
- 플레이어가 장면 구성
- 외계인이 통역기로 직접 확인 ("당신은 ~~를 그린 것입니까?")
- 플레이어가 수정 또는 최종 제출
- 여러 라운드 반복

**평가:** 신뢰 잃은 상태에서 무엇을 보여주는가. 프로파일 보강.

---

### 6.6 Day 5 — 최후 판결

**서사:**
5일간의 모든 데이터가 쌓인 상태. 외계인이 최후 판결을 내리러 온다.

**메카닉:**
- 전체 프로파일 기반 LLM이 최후 질문 하나 생성
- 사물 제시 없음 — 완전 자유 드로잉
- 그림 한 장 제출
- Day 1 프로파일 반전 공개
- 판결

**평가:** Day 2-5 누적 평가 기반. 결말 분기 결정.

---

### 6.7 5일 요약

| Day | 서사 | 메카닉 | 목적 |
|-----|------|--------|------|
| 1 | 첫 만남, 뇌파 보정 | 사물 하나씩 자유 선택 + 뇌파 반응 | 프로파일 생성 |
| 2 | 단방향 해석 완성, 인류 탐구 | LLM 생성 질문 → 연속 그림 스토리텔링 | 프로파일 보강 |
| 3 | 양방향 소통 → CIA 사건 | 자유 대화 + LLM 생성 압박 질문 → 자유 텍스트 대답 | 프로파일 보강 |
| 4 | 신뢰 붕괴, 다시 그림으로 | 사물 2개 → 장면 구성 → 외계인 직접 확인 | 프로파일 보강 |
| 5 | 최후 판결 | LLM 생성 최후 질문 → 자유 드로잉 → 반전 → 판결 | 결말 분기 |

## 6. 피드백 레이어

플레이어에게 전달되는 피드백은 세 채널을 통해 순차적으로 제공된다.

| 레이어 | 타이밍 | 채널 | 해석 여지 |
|--------|--------|------|-----------|
| 1. 외계인 확인 | 제출 전 | 통역기 텍스트 | 낮음 (명확) |
| 2. 외계인 제스처 | 제출 직후 | 3D 애니메이션 | 높음 (모호) |
| 3. 해석기 출력 | 제스처 후 | 터미널 텍스트 | 중간 (불완전, 점점 선명해짐) |

### 7.1 레이어 1 — 외계인 직접 확인

제출 전, 외계인이 통역기를 통해 "당신은 ~~를 표현한 것입니까?"라고 묻는다. 플레이어는 수정하거나 그대로 최종 제출한다.

### 7.2 레이어 2 — 외계인 제스처 반응

표정, 몸짓 등 비언어적 제스처로 즉각적인 감정 반응을 전달. 해석 여지가 있어 추리의 재미 유지. 부관이 옆에서 나레이션으로 보조.

**구현:** 3D 애니메이션. 모델은 Meshy.ai로 생성, 모션 캡처는 FreeMocap(오픈소스) 검토 예정.

### 7.3 레이어 3 — 해석기 출력 (텔레파시 조각)

외계인의 텔레파시를 불완전하게 번역해 터미널에 표시. Day가 진행될수록 해석 품질이 높아진다.

```
Day 1:  ▓▓▓ ∆∆∆ ////// ██ ∞∞∞
Day 2:  ▓▓ SHOW ////// ██ MORE ∞∞
Day 3:  ▓ THIS IS — SHOW ME ∞ STRENGTH
Day 4:  THIS DISPLAY IS — NOTED.
Day 5:  THIS DEMONSTRATION SATISFIES THE DELEGATION.
```
