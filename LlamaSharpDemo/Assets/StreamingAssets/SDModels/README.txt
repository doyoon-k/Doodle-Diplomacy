Place Stable Diffusion model files here for local runtime inference.

Recommended flow:
  1. Open Tools > LLM Pipeline > Setup Wizard
  2. Use the "Stable Diffusion Models" section
  3. Download a preset directly into StreamingAssets/SDModels
  4. Let the wizard auto-apply StableDiffusionCppSettings

Default preset filenames:
  Assets/StreamingAssets/SDModels/stable-diffusion-v1-5-pruned-emaonly-Q4_1.gguf
  Assets/StreamingAssets/SDModels/sd_turbo-f16-q8_0.gguf

You may also place a compatible GGUF model here manually and then use
"Apply Existing Download To Settings" in the Setup Wizard.
